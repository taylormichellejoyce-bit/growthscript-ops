# PMM Partner - Getting Started

A Claude Code plugin that helps Product Marketing Managers build and maintain GTM documentation.

---

## What You Need First

**Claude Code** - This plugin runs inside Claude Code, Anthropic's terminal-based AI assistant.

If you don't have Claude Code yet:
1. Go to [claude.ai](https://claude.ai) and create an account (if you don't have one)
2. Subscribe to Claude Pro or Max
3. Install Claude Code: [docs.anthropic.com/en/docs/claude-code](https://docs.anthropic.com/en/docs/claude-code)

---

## Install PMM Partner (5 minutes)

### Step 1: Download the plugin

Go to: **[github.com/taylormichellejoyce-bit/pmm-partner](https://github.com/taylormichellejoyce-bit/pmm-partner)**

Click the green **Code** button → **Download ZIP**

### Step 2: Unzip and install

1. Find the downloaded ZIP file (probably in Downloads)
2. Double-click to unzip it
3. Open **Terminal** (Cmd + Space, type "Terminal", press Enter)
4. Run these commands:

```
cd ~/Downloads/pmm-partner-main
./install.sh
```

You should see "Installation complete!"

### Step 3: Start using it

1. Open a **new** Terminal window
2. Type `claude` and press Enter
3. Once Claude starts, type: `/pmm-partner:setup`

The setup wizard will walk you through connecting your tools.

---

## Available Skills

| Command | What it does |
|---------|--------------|
| `/pmm-partner:setup` | Connect Claude to your tools (Notion, Linear, etc.) |
| `/pmm-partner:core-products` | Document your products and capabilities |
| `/pmm-partner:releases` | Track what's shipping and GTM readiness |
| `/pmm-partner:icp` | Build ICP and persona documentation |
| `/pmm-partner:competitive` | Track competitors and build battlecards |

---

## Quick Tips

- **You don't need to be technical.** The plugin guides you through everything conversationally.
- **Works with any doc tool.** Notion, Confluence, Google Docs, or even local files.
- **Start with setup.** Run `/pmm-partner:setup` first to connect your tools.
- **Use existing content.** You can drop files into Terminal or paste content - no need to start from scratch.
